--- layout: post title: IE9 and strict JSON checking date: '2011-11-08T13:08:00.000-05:00' author: Bill Schneider tags: modified_time: '2011-11-08T13:08:18.787-05:00' blogger_id: tag:blogger.com,1999:blog-9159309.post-165476981853438622 blogger_orig_url: http://wrschneider.blogspot.com/2011/11/ie9-and-strict-json-checking.html --- Got an error from IE9 complaining about "invalid character". Turned out this was because the JSON response string had control characters (newlines) in it. These need to be escaped.

Other browsers will let you get away with it, but IE9 is particularly strict and jQuery doesn't check the JSON response string before handing it off to JSON.parse.
